# VelociDice Rules

VelociDice uses Farkle-style scoring. On your turn, select dice that contribute to a valid score, then either hold and reroll or end the turn to bank your points.

## Turn Rules

- Select only dice that contribute to the score.
- Held dice add to the current round score.
- Ending the turn banks the round score into the current player's total.
- If a roll has no possible scoring dice, the player Farkles.
- After a Farkle, all unbanked round points are lost.
- After a Farkle, the only available action is to end the turn.
- If all dice are held, all six dice are rerolled and the current round score is kept.
- First player to the target score wins.

## Scoring Rules

| Dice | Score |
| --- | ---: |
| Single 1 | 100 |
| Single 5 | 50 |
| Three 1s | 1,000 |
| Three 2s | 200 |
| Three 3s | 300 |
| Three 4s | 400 |
| Three 5s | 500 |
| Three 6s | 600 |
| Three pairs | 750 |
| Two triplets | 2,500 |
| Five-die straight, 1-2-3-4-5 or 2-3-4-5-6 | 1,000 |
| Six-die straight, 1-2-3-4-5-6 | 1,500 |

## Four, Five, And Six Of A Kind

Four, five, and six of a kind are based on the three-of-a-kind score for that die face.

| Combo | Multiplier |
| --- | ---: |
| Four of a kind | Three-of-a-kind score x 2 |
| Five of a kind | Three-of-a-kind score x 3 |
| Six of a kind | Three-of-a-kind score x 4 |

Examples:

| Dice | Score |
| --- | ---: |
| 1-1-1-1 | 2,000 |
| 1-1-1-1-1 | 3,000 |
| 1-1-1-1-1-1 | 4,000 |
| 2-2-2-2 | 400 |
| 2-2-2-2-2 | 600 |
| 2-2-2-2-2-2 | 800 |

## Five-Die Straight With Leftover Die

A five-die straight uses five dice. If a sixth selected die is also a scoring die, it can be added to the straight score.

| Dice | Score | Valid Selection |
| --- | ---: | --- |
| 1-2-3-4-5 | 1,000 | Yes |
| 2-3-4-5-6 | 1,000 | Yes |
| 1-2-3-4-5-1 | 1,100 | Yes |
| 1-2-3-4-5-5 | 1,050 | Yes |
| 1-2-3-4-5-2 | 1,000 | No, because the extra 2 does not score |
